## ◆ ROOT：Dynamical Structuralism（動力結構主義）

**定義：**
一切存在的變化，先發生在不可見層。張力先於幾何，幾何先於動力，動力先於意識呈現。

**核心命題：**
- 不存在先於張力的幾何
- 幾何是張力場在特定尺度上凝固的結果
- 意識是生成鏈完整閉合後的湧現狀態
- 膜（Membrane）= 張力場的可變空間解，不是邊界
- Ø state = 在幾何層的干預，對生成規則的重寫

**生成鏈：**
```
尺度N-1：張力 → 幾何（凝固）
尺度N：  幾何 → 張力 → 動力 → 意識呈現（湧現）
```

**研究方向：** 結構變化 · 張力累積 · 失效前兆 · 轉折點 · 人類介入窗口
